## 🚀 Instalación y Ejecución

### Requisitos Previos

- **Node.js** (versión 16 o superior)
- **npm** (versión 8 o superior) o **yarn**
### Pasos de Instalación

```bash

# 1. Navegar al directorio del mockup
cd RutaCEA/mockup4_220626/
# 2. Instalar dependencias
npm install
# 3. Iniciar el servidor de desarrollo
npm run dev
```
### Comandos Disponibles

```bash

npm run dev      # Servidor de desarrollo en http://localhost:3000/
npm run build    # Construir para producción (carpeta dist/)
npm run preview  # Previsualizar la versión de producción
npm run lint     # Ejecutar ESLint para verificar código

```
